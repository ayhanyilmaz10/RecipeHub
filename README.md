
# RecipeHub

Bu proje, TheMealdb API kullanılarak hazırlanmış modern bir Yemek tarifi uygulamasıdır. Yemeğin ana vatanını, görselini ve ne kadar sürede hazırlandığını gösterir.

## 🛠 Teknolojiler

- ⚛️ React
- ⚡ Vite
- 🎨 Tailwind CSS
- 🧹 ESLint
- 🟦 TypeScript
